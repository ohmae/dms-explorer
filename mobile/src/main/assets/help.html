What is this app?

This app is a DLNA player.

You can view the contents of DLNA servers on the same network.

You will need to prepare a DLNA server in your own environment.

For more information on what DLNA is, please refer to the following website.

DLNA (Wikipedia)

What if I can't find the DLNA server?

Please check the following

  • WiFi or wired LAN connection is required. It will not work over a cellular line.
  • Make sure that a DLNA server is connected to the connected network.
    Also, make sure that the DLNA server is enabled in the settings of your server, etc.
    For information on how to set this up, contact the server manufacturer.
